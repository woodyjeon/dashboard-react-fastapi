# 포트폴리오 섹션

## 개요

포트폴리오는 캐러셀 형태로 프로젝트를 소개합니다. 중앙에 큰 카드가 있고 좌우에 이전/다음 프로젝트 미리보기가 보입니다.

## 탐색 방법

- 좌우 화살표 버튼으로 이동
- 하단 인디케이터(1/N)로 현재 위치 확인
- 카드 클릭 시 프로젝트 상세 URL로 이동

## 등록된 프로젝트

### wjeon Dashboard
- 분야: 자동차
- 설명: 뉴스, 포트폴리오, RAG 챗봇을 한 곳에 모은 개인용 대시보드
- 기술: React, Vite, FastAPI

### RAG Knowledge Assistant
- 분야: AI
- 설명: 사내 문서를 임베딩해 자연어로 검색하는 RAG 기반 어시스턴트
- 기술: LangChain, Vector DB, Python

### Realtime News Pipeline
- 분야: Industrial
- 설명: RSS와 외부 API를 수집해 실시간으로 정제하는 뉴스 파이프라인
- 기술: FastAPI, Async, ETL

### SMK Agent Platform
- 분야: Automation
- 설명: 여러 도구를 연결해 작업을 자동 실행하는 에이전트 플랫폼
- 기술: Agent, Orchestration

### Design System Kit
- 분야: Frontend
- 설명: 일관된 디자인 토큰으로 만든 재사용 가능한 컴포넌트 디자인 시스템
- 기술: CSS, Tokens, A11y
