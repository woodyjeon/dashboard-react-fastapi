# Woody Jeon Dashboard 소개

Woody Jeon Dashboard는 전우열(Woody Jeon)의 개인용 포트폴리오 대시보드입니다.

## 제공 기능

- **실시간 뉴스 피드**: 네이버 IT/과학 섹션 기사를 크롤링해 카드 형식으로 자동 가로 스크롤합니다.
- **포트폴리오 캐러셀**: 프로젝트를 중앙 대형 카드와 좌우 미리보기로 탐색할 수 있습니다.
- **RAG 챗봇**: 대시보드 지식 베이스를 검색해 OpenAI 모델로 답변합니다.
- **SMK Agent**: 외부 에이전트 앱을 링크 카드 또는 iframe으로 연결합니다.

## 기술 스택

- 프론트엔드: React 19, Vite 8, 커스텀 CSS, lucide-react
- 백엔드: FastAPI (Python), LangChain, OpenAI API, FAISS 벡터 스토어
